Arbeit Pro
==========

Coloca aqui los archivos de la tipografia corporativa para que la demo HTML
los embeba automaticamente via @font-face:

  ArbeitPro-Regular.woff2   (peso 400)
  ArbeitPro-Medium.woff2    (peso 500)
  ArbeitPro-Bold.woff2      (peso 700)

Formatos aceptados: .woff2 (preferido), .woff, .otf o .ttf.
Si cambias la extension, actualiza las rutas en el bloque @font-face de
scratchpad/wa-comparativa.html.

Mientras no esten los archivos, la demo usa un fallback geometrico
(system-ui / Segoe UI / Arial), por lo que se ve correcta igual.

Para el PPTX (build_deck.js) la fuente debe estar INSTALADA en Windows con el
nombre de familia "Arbeit Pro" para que PowerPoint la renderice al abrir el deck.
